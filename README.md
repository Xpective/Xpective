## Main Project

### Inpinity
Inpinity is a growing on-chain game and world-building ecosystem.

Core layers include:

- **Inpinity NFT** — Pyramid block ownership
- **ResourceToken** — resource economy
- **Farming** — on-chain production
- **Pirates** — raids and strategy
- **Mercenary** — defense and protection
- **Partnership** — alliances and cooperative mechanics
- **Pitrone / INPI** — ecosystem currencies
- **Future Inpinity City** — plots, buildings, crafting, trading, expansion

The long-term goal is to connect blockchain logic with a living game world and a stronger strategic economy.
